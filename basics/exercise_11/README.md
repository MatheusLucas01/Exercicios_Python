# Fluxo do código

✔ 1. O programa solicita ao usuário que digite um número.

✔ 2. Se o usuário digitar algo que não seja um número, uma mensagem de erro será exibida, pedindo para tentar novamente.

✔ 3. O programa verifica o número inserido:

    Se for maior que 0, exibe a mensagem: O número informado é positivo.
    Se for menor que 0, exibe a mensagem: O número informado é negativo.
    Se for igual a 0, exibe a mensagem: None (nenhum número).
